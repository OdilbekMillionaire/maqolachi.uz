# MAQOLACHI - AI Academic Writing Platform

Ilmiy maqolalaringizni professional darajada yarating.

## Technologies

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Zustand (state management)
- Supabase (backend)
- Framer Motion (animations)

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```
